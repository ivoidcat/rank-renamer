# Rank Renamer

Win98 风格的批量图片重命名工具，基于 Flask Web 界面。

## 功能

- 缩略图网格预览
- 拖拽调整顺序
- 多种排序方式（名称、修改时间、创建时间、文件大小、扩展名）
- 自定义命名规则（前缀、起始编号、步长、补零位数、后缀）
- 重命名预览窗口
- 输出到独立文件夹，不覆盖原文件

## 安装

需要 [uv](https://docs.astral.sh/uv/) 包管理器：

```bash
# macOS / Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

# 克隆项目
git clone https://github.com/ivoidcat/rank-renamer.git
cd rank-renamer

# 安装依赖
uv sync
```

## 使用

```bash
uv run rename.py
```

浏览器会自动打开 `http://127.0.0.1:5000`。

### 操作步骤

1. **选择文件夹** — 在起始页输入包含图片的文件夹路径，点击"打开文件夹"
2. **排序** — 左侧面板选择排序方式（原始名称、名称、修改时间、创建时间、文件大小、扩展名）
3. **拖拽调整** — 在右侧网格中拖拽图片到目标位置，微调顺序
4. **设置命名规则** — 左侧面板配置前缀、起始编号、步长、补零位数、后缀，实时查看示例
5. **预览** — 点击"预览结果"弹出新窗口，查看每张图片对应的新文件名
6. **执行** — 点击"执行重命名"，文件会被复制到源文件夹同级的 `output/` 目录，原文件不受影响

### 命名规则示例

| 前缀 | 起始 | 步长 | 位数 | 后缀 | 结果 |
|------|------|------|------|------|------|
| `rank_` | 1 | 1 | 0 | | `rank_1.png, rank_2.png, ...` |
| `level` | 0 | 1 | 3 | | `level000.png, level001.png, ...` |
| `img_` | 1 | 2 | 0 | `_final` | `img_1_final.png, img_3_final.png, ...` |
